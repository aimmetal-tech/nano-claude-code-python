# Contributing to nano-claude-code

这是一个教学项目，非常欢迎各位参与进来！

## 在贡献之前

所有贡献必须基于一个Open的github issue，除非更改是针对小的错误修复、拼写错误更正、轻微措辞改进或不改变功能的简单类型修复。

对于功能和贡献：

- 首先检查issues，寻找类似的问题
- 如果你的想法是新的，请创建一个新的issue
- 在开始实施前等待@nano-claude-code-maintainer的批准
- 获得批准后，您可以在我们社区的帮助下进行PR的工作

**未经Issue批准的PR可能会被无条件关闭**

## 第一次参与贡献？

欢迎查阅：[Good-First-Issues](https://github.com/TIC-DLUT/nano-claude-code/issues?q=is%3Aissue%20state%3Aopen%20label%3A%22good%20first%20issue%22)

这是我们专为新的贡献者设计的issus，跟着每日的学习内容完全可以完成内容，同时带有一个实现指导、难度、和推荐完成进度

**Good-First-Issues拒绝一切AI代码提交！我们将会拒绝违反该规定的贡献者的全部PR！**

## 贡献流程

1. 认领或者新开一个Issue，得到维护者许可
2. Fork 本项目
3. 提交你的修改
4. 开启一个 Pull Request
5. @nano-claude-code-maintainer 进行审核

贡献过程中欢迎与我们的维护者沟通
